# File Upload Previewer

Vanilla JavaScript file uploader with drag & drop, validation, live previews, reordering, and mock upload.

## Features
- Drag & drop / browse / paste
- Image & video thumbnails, PDF icon
- Type & size validation (5 MB/file, 20 files)
- Reorder ↑ ↓, remove, clear all
- Mock upload with `FormData`
